### inline

- 使元素变成行内元素，拥有行内元素的特性，可以和其它行内元素共占一行，不会独占一行；不能改变 height、weight，大小由内容撑开，

- 可以使用 padding，margin 上下不生效

### block

- 使元素变成块级元素，独占一行，在不设置自己的宽度的情况下，块级元素会默认填满父级元素的宽度
- 能够改变元素的 height、weight
- 可以使用 padding，margin

### inline-block

- 可以和其他元素共占一行
- 可以设置 weight、height
- 可以设置 padding、margin

### none

不显示

### flex

弹性盒子布局
